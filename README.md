# Event Management Application

Application web de gestion d’événements développée avec Spring Boot et Vaadin.  
Elle permet la création, la gestion et la consultation d’événements avec authentification des utilisateurs et gestion des rôles.

---

## Technologies utilisées

- Langage : Java 17  
- Framework Backend : Spring Boot 3.5.8  
- Framework Frontend : Vaadin 24.9.5  
- ORM : Hibernate (implémentation JPA)  
- Base de données : H2 (embedded)  
- Build : Maven  
- Serveur : Tomcat embarqué  

---

## Prérequis

- Java JDK 17 installé
- Maven 3.9+
- Un IDE (IntelliJ IDEA recommandé, Eclipse possible)
- Un navigateur web moderne (Chrome, Firefox, Edge)
